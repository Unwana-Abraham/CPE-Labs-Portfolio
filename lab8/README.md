# Lab Project: TO-DO LIST APP
**Reg No:** 22/EG/CO/1669

## Reflection
I learned to integrate dynamic DOM manipulation with `localStorage` for data persistence. I also practiced using CSS Media Queries to ensure a responsive layout across different screen sizes.

## Challenges
The main challenge was making a desktop-designed calendar fit a mobile screen. I solved this by switching to percentage-based widths and vertical flex stacking in the CSS.
