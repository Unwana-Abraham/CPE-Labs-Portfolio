# Lab 7: Events and Interactivity in JavaScript


## Reflection
I learned how to use event listeners to make web elements respond to user actions. I moved from writing static code to managing "state" (like the counter) and interacting with the DOM.

## Challenges
The main challenge was handling the browser's default behaviors, such as page refreshes on form submission. Troubleshooting syntax errors in a mobile editor also required extra attention to detail regarding symbols and closing brackets.
