## Lab 5: Reflection - JavaScript Basics

### What I Learned
In this lab, I learned how to move from static design to functional logic. The most significant takeaway was understanding how to use **Variables** to store data and **Functions** to perform reusable actions. I also learned the importance of **Separation of Concerns** by moving my code into an external `script.js` file, which makes the project more organized and professional.

### Challenges Faced
The biggest challenge was understanding the logic of **Loops** and how the counter (`i`) relates to the array (`fruits`). Initially, it was confusing how a simple number could "point" to an item in a list. However, by breaking it down into a "step-by-step" process, I now understand that the loop is essentially a tool for automated data processing.
