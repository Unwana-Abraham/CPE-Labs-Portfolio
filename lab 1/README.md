# Lab 1: Introduction to HTML & Version Control

## Project Overview
This was my first project in the CPE Web Development series. The goal was to establish a solid foundation in HTML syntax and set up a professional workflow using mobile development tools and GitHub.

## Key Features
* **Basic HTML Tags:** Implemented headings (`<h1>`-`<h6>`), paragraphs (`<p>`), and lists (`<ul>`/`<ol>`).
* **Hyperlinks & Images:** Learned how to connect pages and display visual content.
* **Environment Setup:** Configured **Spck Editor** on Android and linked it to **GitHub** for version control.

## Technical Reflection
This lab taught me the importance of "Tags" and "Attributes." Learning to use Git on a mobile device was a significant milestone, as it allows me to manage my engineering projects from anywhere.
