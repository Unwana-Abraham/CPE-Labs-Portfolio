# Lab 4: Responsive Web Layouts 

## Project Overview
This project demonstrates the transition from Flexbox to **CSS Grid** for complex layouts and the implementation of **Responsive Design** using Media Queries.

## Technical Features
* **CSS Grid Layout:** Used `grid-template-columns` with fractional units (`1fr`) to create a balanced 3-column structure.
* **Flexbox Navigation:** Implemented a top navigation bar using `display: flex` and `justify-content: space-between`.
* **Responsive Breakpoints:** Added a `@media` query for screens under `600px` to stack content vertically for mobile users.
* **Interactivity:** Applied `:hover` pseudo-classes with `transform: scale()` and `transition` for a smooth user experience.

## How to View
1. Open `lab4.html` in any modern web browser.
2. Resize the window or rotate your mobile device to see the Grid adapt from 3 columns to 1 column.
