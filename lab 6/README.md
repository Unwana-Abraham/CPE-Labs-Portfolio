## 📝 Lab 6: Reflections & Challenges

### Key Learning Moments
* **DOM Navigation:** Mastered the difference between selecting by ID (`getElementById`) and by class (`querySelector`).
* **The "Input-to-Output" Bridge:** Successfully implemented logic to fetch user data using `.value` and display it dynamically using `.textContent`.
* **Efficient State Toggling:** Learned to use `.toggle()` to manage UI changes (like show/hide) with minimal code.

### Challenges Faced
* **Syntax Strictness:** I encountered issues where a lowercase 'l' in `classList` and missing quotes in a query selector caused the script to fail. This reinforced the importance of precise syntax in JavaScript.
* **Debugging with the Console:** During Activity 5, I faced an "Unexpected Token" error. Troubleshooting this taught me how a single punctuation error can halt the execution of an entire script.
* **Error Handling Logic:** Building a "Safety Net" using `if/else` to check for the existence of elements before manipulating them to prevent application crashes.
